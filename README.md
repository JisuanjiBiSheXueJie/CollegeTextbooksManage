# 高校教材管理系统
## 项目介绍
基于springboot + mybatis + shiro + jwt + layui的用于高校管理教材的系统。
## 项目主要功能
* 教材信息管理。
* 教材库存管理，包括入库、出库、分发领取记录管理。
* 教材供应商信息管理。
* 学生、班级、学院等信息管理。
* 教学计划管理。
## 项目特色
* 异步发送邮件用于用户修改密码。
* 增加或修改教材信息时上传教材封面以及预览功能。
* 使用了jwt+shiro进行前后端分离的权限管理和单点登录功能。
